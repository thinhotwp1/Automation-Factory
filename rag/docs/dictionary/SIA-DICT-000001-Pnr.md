## SIA-DICT-000001: Definition: Passenger Name Record (PNR)
- **Business Meaning**: The unique Booking Reference code for SIA passengers.
- **Format Rule**: MUST match the regex pattern `^[A-Z]{2}-\d{5,10}$` (e.g., `SQ-12345`).
